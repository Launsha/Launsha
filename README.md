## Siapa Saya

Developer indie yang membangun web app dan Android APK **sepenuhnya dari smartphone** — Vivo Y91I, 2GB RAM. Tidak ada laptop. Tidak ada workstation. Hanya kode, kreativitas, dan koneksi internet.

Stack utama: **React + TypeScript + Tailwind CSS + shadcn/ui + Capacitor**, di-deploy ke APK via GitHub Actions cloud build. Semua UI dalam **Bahasa Indonesia**, semua desain **dark minimalist**, semua performa dioptimasi untuk perangkat low-end.

Selain developer, saya juga seorang **penulis novel fantasy** dan **peneliti industri webtoon/anime** dari perspektif Indonesia.

## Cara Kerja

Tidak ada IDE desktop. Tidak ada emulator lokal.

1. Desain & build di **Lovable** (React + TypeScript + Tailwind)
2. Push ke **GitHub** secara otomatis
3. **GitHub Actions** compile APK di cloud (Java 21, `actions/upload-artifact@v4`)
4. Download APK langsung ke Android
5. Test di perangkat asli

Semua dari layar 6 inci.

## Filosofi Build

> Perangkat low-end bukan batasan. Itu constraint desain.

Setiap app yang saya buat harus:
- Jalan smooth di 2GB RAM
- Tidak pakai `backdrop-filter` (GPU killer di perangkat lama)
- Animasi hanya `transform` dan `opacity`
- UI tetap gelap, bersih, dan fungsional
- Offline-first kalau memungkinkan
